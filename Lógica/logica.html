Trabalhando com Variáveis

Variáveis são usadas para armazenar valores que podem ser utilizados e manipulados ao longo do programa. Elas são essenciais para o funcionamento de qualquer linguagem de programação.

1. O que são variáveis?

Variáveis são espaços nomeados na memória do computador que armazenam dados. Elas permitem que reutilizemos e modifiquemos valores durante a execução de um programa. São fundamentais para criar programas dinâmicos.

2. Tipos de variáveis em JavaScript

JavaScript trabalha com diferentes tipos de dados:

  • String: textos, como "Olá Mundo"
  • Number: números, como 10, 2.5, -3
  • Boolean: verdadeiro (true) ou falso (false)
  • Null: valor nulo proposital
  • Undefined: variável declarada, mas sem valor atribuído
  • Object: estrutura de dados com propriedades
  • Array: lista ordenada de valores

3. Declaração de Variáveis

Podemos declarar variáveis em JavaScript de três formas:

  • var: forma antiga, escopo global ou de função.
  • let: escopo de bloco, recomendada para variáveis que mudam.
  • const: escopo de bloco, usada para valores constantes.
var nome = "Carlos";
let idade = 28;
const cidade = "Curitiba";

4. Boas Práticas

  • Use let e const em vez de var
  • Escolha nomes significativos para as variáveis
  • Evite abreviações desnecessárias
  • Inicialize as variáveis sempre que possível
  • Use const quando o valor não será reatribuído

5. Exemplos Resolvidos

Exemplo 1: Concatenar texto e número

let nome = "Ana";
let idade = 25;
console.log(nome + " tem " + idade + " anos.");

Exemplo 2: Soma de dois números

let a = 10;
let b = 20;
let soma = a + b;
console.log("A soma é: " + soma);

Exemplo 3: Alterar valor de uma variável

let contador = 1;
contador = contador + 1;
console.log("Contador: " + contador);

6. Atividades para Praticar

  • Declare uma variável com seu nome e imprima no console.
  • Crie duas variáveis com números e mostre a soma.
  • Utilize const para declarar um valor fixo e tente alterá-lo.
  • Declare uma variável booleana e imprima seu valor.
  • Crie um programa que calcule a média de duas notas.
  • Crie um objeto com informações sobre um aluno (nome, idade, curso).
  • Crie um array com 5 frutas e mostre a terceira fruta.
